# Customer Segmentation Dashboard Notes

## Dashboard Objective

This dashboard explains customer value using RFM segmentation. It helps identify which customer groups deserve retention, reactivation, or onboarding attention.

## Views Included

- Customers by Segment: shows the size of each RFM group.
- Revenue by Segment: shows which segments contribute the most monetary value.
- Avg Recency by Segment: shows how recently each customer group purchased.
- Customer Value Detail: ranks customers by segment and value for follow-up targeting.

## Business Story

Champions and loyal customers represent the strongest retention opportunity because they purchase recently and contribute high value. At-risk and hibernating customers need reactivation campaigns, especially when their historical monetary value is high. New customers should receive onboarding campaigns to encourage repeat purchase behavior.

## Suggested Talking Point For CV Or Interview

Built an RFM customer segmentation dashboard in Tableau to compare customer count, revenue contribution, recency behavior, and high-value customer targets across lifecycle segments.
